## 7. Размер главной страницы (архитектурно)

Главная /tashkent весит ~1MB HTML. Это тяжело для мобильных.

**Проблема:** Все продукты загружаются в HTML при SSR.

**Возможные решения (обсудить с разработчиками):**
- Ленивая загрузка категорий (каждая категория подгружается при скролле)
- Убрать `__NEXT_DATA__` JSON из HTML (там ~900KB!) — загружать продукты через API
- Включить streaming SSR в Next.js

**Когда:** отдельно от SEO-фиксов, это архитектурная задача.
